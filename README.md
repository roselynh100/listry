## How It Works

Listry analyzes the user's inputted grocery list and postal code, then uses the Google Maps API to locate the nearest grocery stores nearby. Behind the scenes, grocery prices webscraped from various online retailers â€” such as Walmart and Longos â€” are compiled. Finally, the total price at each store close by is presented to the user, allowing them to make an informed decision on where they want to do their groceries.
